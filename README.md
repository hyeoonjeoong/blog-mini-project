# blog-mini-project

### 📌 메인페이지

- 전체 포스팅 조회
- Grid 반응형 적용
- 클릭 시 포스팅 상세로 이동 (useNavigate를 사용해 state값 전달)
- 썸네일의 경우 이미지 데이터를 랜덤으로 보여주도록 적용.
- 블로그 데이터의 경우 json 파일을 생성, api파일을 분리해 각 요청 적용.
